========================================================================
          SIMULATION README: AGENT BASED SIMULATION FOLDER
========================================================================

This folder contains source files for the agent based simulation.


========================================================================
		             Files
========================================================================

simulation

This is the top-level class of the simulation. After creating an instance
of it, by loading design, text and a map file from: \simulation\Loadable
\Sim Designs, it can be run by using its run() function.


node, location, system, road, car, MOE, route, lights, varClock

These are simulation objects and for a newcomer, are unlikely to be
needed to be understood or interacted with.

========================================================================
                  SIMULATION README: TOP LEVEL FOLDER
========================================================================

This folder contains source files for the top level of the project.


========================================================================
		             Files
========================================================================

main

This is the top level class of the project. 
In this file there exists commented code that demos what can be done.


functions

Useful functions to be called by main.


files

locations of loadable files (such as controllers and simulation designs).


Config

Simulation settings (defined by macros).


stdafx

Pre-compiled headers. Should be no reason to edit these.


targetver

Includes important libraries for the VS C++ environment.
Should be no reason to edit this.
